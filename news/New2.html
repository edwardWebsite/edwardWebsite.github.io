The collatz conjecture

The collatz conjecture is quite simple. You start with any number. If the number is even, you divided it by 2. If the number is odd, you multiply it by 3 and add 1. Then, you do the same process with the resulting number. The collatz conjecture states that if you do these simple operations, you will always end with the number 1.


Example

We start with number 14. The sequence would be:
14, 7, 22, 11, 34, 17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1

No one ever proves the conjecture.
